RESETTY(1) General Commands Manual RESETTY(1)

NAME

resetty - heuristic terminal reset clone

SYNOPSIS

resetty [TTY]

DESCRIPTION

resetty resets ttys in a dynamic fashion if no tty is specified. It employs some heuristics to find the attached terminal. (It parses the "proc" directory for example). Otherwise it just resets the specified tty, if permissible by the operating system.

EXIT STATUS

Returns 0 on successfully attempted resets, 1 on failed setup routines.

FILES

/proc/

NOTES

Currently resetting a "remote" tty causes the terminal to just display a blank screen with just a (blinking) cursor. This is completely normal. Just press "<ENTER>" / "<CTRL>+<J>" or any other shell reset key to restore your prompt. There seems nothing to be done about it. Otherwise your terminal is properly reset.

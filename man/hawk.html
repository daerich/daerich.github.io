NAME

hawk - watch with environment heuristics

SYNOPSIS

hawk [OPTION..] [COMMAND]

DESCRIPTION

hawk runs a command for n seconds (2 seconds by default) and displays its output, updated every interval. The shell is determined dynamically from the environment, which can be overriden by its environment variable or a command line switch ( -s ).

OPTIONS

-n, --interval NUMBER Set refresh interval in seconds(default is 2 seconds).

-s, --shell PATH Set specific shell to be used instead.

Note: If the shell doesn't support non interactive shell calls, a POSIX compliant shell is expected at command "sh".

--force-stdout Force Standard Output on display

Note: Normally hawk dynamically determines the output to be displayed, based on the command's exit status. If your command always prints to STDOUT regardless of its exit status use this option.

-v, --version Print version information and exit successfully.

ENVIRONMENT

$SHELL

The shell to be used by default(usually set by your initscripts).

$PATH

Relied upon for shell resolution(for portability reasons)

RETURN STATUS

Almost always exits successfully, unless you specify invalid options, numbers, commands. If you play it right, it will exit successfully...
